# Elmish-fss-template

A simple F# elmish template with:
- F# 5
- Fable 3
- Paket
- Fss
- Elmish

This template implements the basic todo sample from https://reactjs.org/.
A simple styling branch on branch `styling`

## How to run:
```
dotnet tool restore
dotnet paket install
npm install
npm start
```
Go to localhost:1234
